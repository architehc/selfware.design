# Weather CLI — Bug Fixing Demo

A small Rust project with a few bugs for you to find and fix using selfware.

## Try this command

```bash
selfware run "find and fix all bugs, then run cargo test to verify"
```

## Known issues

This project has 3 intentional bugs. Can selfware find them all?

1. Wrong temperature conversion formula
2. Off-by-one error in average calculation
3. Unused import warning
