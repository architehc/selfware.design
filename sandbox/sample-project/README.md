# Weather CLI — Selfware Trial Project

A small Rust project with a few bugs for you to find and fix using selfware.

## Try these commands

```bash
# Analyze the project for issues
selfware analyze .

# Ask selfware to fix the bugs
selfware run "fix the bugs in this project"

# Run the tests to see failures
selfware run "run cargo test and fix any failing tests"
```
